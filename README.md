# Connectly

Connectly is a social media web application where users can create accounts, share posts, follow other users, like posts, and communicate through comments.

## Features

- User registration and login
- Secure password hashing
- User profiles
- Follow and unfollow users
- Followers and following counts
- Create posts
- Like and unlike posts
- Comment on posts
- Discover other users
- Responsive social media interface

## Technologies Used

### Frontend
- React
- Vite
- CSS

### Backend
- Python
- Flask
- SQLite
- SQLAlchemy
- Flask-CORS
